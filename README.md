# 📄 AI-Powered Document Search Engine

> A smart Generative AI-based document retrieval and question-answering system using Retrieval-Augmented Generation (RAG), FAISS, and Google Gemini.

 


---

## 🧠 Overview

The **AI-Powered Document Search Engine** allows users to upload documents, search them using both keyword and semantic search, and ask natural language questions. The system retrieves contextually relevant information using vector embeddings and generates accurate answers using Google Gemini LLM.

This project demonstrates real-world **RAG (Retrieval-Augmented Generation)** architecture.

---

## ✨ Key Features

### 🔐 Authentication
- Secure login system for user access control

### 📁 Document Management
- Upload multiple documents
- Download stored documents

### 🔎 Intelligent Search
- File name-based search
- Semantic search using FAISS vector database

### 💬 AI Question Answering
- Ask questions in natural language
- Context-aware responses using RAG pipeline
- Powered by Google Gemini API

### 📊 User Experience
- Search history tracking
- Clean Streamlit UI
- Fast retrieval using embeddings
