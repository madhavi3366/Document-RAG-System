# 📂 AI-Powered Document Search Engine

> An intelligent Generative AI-based document retrieval and question-answering system powered by **Retrieval-Augmented Generation (RAG)**, **FAISS**, **Google Gemini**, and **Streamlit**.

---

## 🚀 Project Overview

The **AI-Powered Document Search Engine** is a smart document retrieval system that enables users to efficiently search, retrieve, and interact with large collections of documents.

Users can search documents by file name, perform semantic search using vector embeddings, and ask natural language questions to obtain context-aware answers generated by Google's Gemini Large Language Model.

This project demonstrates a real-world implementation of the **Retrieval-Augmented Generation (RAG)** architecture used in modern enterprise AI applications.

---

## ✨ Features

### 🔐 Secure Authentication

* User login authentication system
* Protected access to documents and search functionalities

### 📁 Document Management

* Upload and manage multiple documents
* Support for PDF and CSV documents
* Download documents directly from the application

### 🔎 Intelligent Search

* File name-based document search
* Semantic search using vector embeddings
* Fast similarity retrieval using FAISS

### 💬 AI-Powered Question Answering

* Ask questions in natural language
* Context-aware answers using RAG pipeline
* Powered by Google Gemini LLM
* Source document identification for transparency

### 📊 CSV Analytics

* Analyze structured CSV datasets
* Retrieve statistical insights
* Answer dataset-related questions dynamically

### 📝 User Experience

* Search history tracking
* Interactive and responsive Streamlit UI
* Clean and user-friendly interface

## 🏗️ System Architecture

```text
User Query
     │
     ▼
Streamlit User Interface
     │
     ▼
Document Search Engine
     │
     ├── File Name Search
     │
     └── Semantic Search
              │
              ▼
      HuggingFace Embeddings
              │
              ▼
         FAISS Vector Store
              │
              ▼
      Relevant Document Chunks
              │
              ▼
     Google Gemini LLM
              │
              ▼
      Context-Aware Response
```
